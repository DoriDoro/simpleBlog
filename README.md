## Installation process:
1. $ clone the project: https://github.com/DoriDoro/simpleBlog
2. $ cd simpleBlog
3. $ python -m venv venv
4. $ . venv/bin/activate
5. $ pip install -r requirements.txt
6. $ cd simple_blog
7. $ python3 manage.py migrate
8. $ python3 manage.py runserver
